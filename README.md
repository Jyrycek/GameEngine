# Simple Game Engine

Lightweight C++ 3D graphics engine developed for real-time rendering and educational purposes. It uses OpenGL and supports loading models, shaders, lighting, and scene management.


## Controls

- **W / A / S / D** – Move the camera (forward, left, backward, right)
- **Mouse Movement** – Look around (first-person camera control)
- **SPACE** – Switch between different scenes

## Dependencies

- GLEW
- GLM
- Assimp
- SOIL
